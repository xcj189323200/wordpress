== 描述 ==

这款SEO插件可以为您的WordPress网站首页、文章页、普通页面、分类目录、标签添加自定义SEO设置选项: 标题(title)、 描述(description)、关键字(keywords）。正确使用该SEO插件可以有效地提升您的WordPress网站在搜索引擎的排名和索引量。

== 安装 ==

1. 进入WordPress管理后台 - 插件 - 安装插件 - 添加插件 - 上传插件。
2. 选择已下载到本地的zhutibaba-seo.zip压缩包。
3. 点击现在安装。
4. 点击激活。
5. 首页SEO设置，请进入WordPress管理后台 - 设置 (Settings) - 常规，进行设置。编辑文章、页面、分类目录、标签时，即可设置相应的SEO选项。
